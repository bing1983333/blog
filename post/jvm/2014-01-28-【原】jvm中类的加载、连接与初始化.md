【原】jvm中类的加载、连接与初始化
====
直接看一个例程
```java
public class JvmClassTest {  
    public static JvmClassTest OBJ = new JvmClassTest();  
    public static int A;  
    public static int B = 0;  
    //public static JvmClassTest OBJ = new JvmClassTest();  
      
    static {  
        System.out.println("A:" + A);  
        System.out.println("B:" + B);  
    }  
      
    public JvmClassTest() {  
        A++;  
        B++;  
    }  
      
    public static void main(String[] args) throws Exception {  
        JvmClassTest.B = -1;  
    }  
}  
```
